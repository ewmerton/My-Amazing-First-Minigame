# Minigame

## Description

A minigame for SDL testing

## Key Features

You are a turtle, and you can swim like a turtle... and you can shoot bubbles to a jellyfish 

## Controls

Move: Up, Down, Left and Rigth arrows

Shoot: Space bar

Exit game: Esc
