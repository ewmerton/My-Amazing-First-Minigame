# My-Amazing-First-Minigame

## Description

A shoot'em up minigame based on the classic "Spacewar!"

## Key Features

## Controls

 - A, W, S and D to move
 - SPACE to shoot
 - F1 to toggle God Mode
 - ESC to exit
